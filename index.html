Minhas SKILLS

HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar e estruturar páginas da web. Utiliza tags para definir elementos como textos, links, imagens e outros conteúdos.

CSS

CSS (Cascading Style Sheets) é uma linguagem utilizada para descrever a apresentação de páginas web, incluindo layout, cores, fontes e outros aspectos visuais. Ele trabalha em conjunto com o HTML, aplicando estilos aos elementos através de seletores.

JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade e dinamismo às páginas web. Executada no navegador, pode manipular elementos HTML e CSS, responder a eventos do usuário, validar formulários e comunicar-se com servidores para carregar ou enviar dados
